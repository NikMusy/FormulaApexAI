# Formula Apex AI 🏎️🧠

Самообучающийся ИИ-автопилот для Roblox-плейса **Formula Apex Racing**.
Смотрит на экран игры, распознаёт трассу, **учится у тебя** и потом **едет сам** —
по гоночной линии, с батареей (ERS), запоминая всю трассу и улучшая время круга.

Чистый Python, без тяжёлых зависимостей: компьютерное зрение на `numpy`,
захват окна на `mss`, ввод в игру через WinAPI `SendInput`.

---

## Возможности

- 🎯 **Захват только окна Roblox** (не весь экран) — находит окно через WinAPI.
- 👁️ **Самокалибрующееся зрение**: сам определяет цвет асфальта плейса и видит
  трассу целиком (ближняя/средняя/дальняя зоны + кривизна поворота).
- 🕹️ **Управление как в игре**: газ = ЛКМ, тормоз/назад = ПКМ, батарея (ERS) = G,
  руль = абсолютное позиционирование курсора (не «уезжает»).
- 🧠 **Обучение тремя способами**:
  - **Фоновое** (всегда): пока ты играешь сам — молча запоминает твою манеру.
  - **Запись (F5)**: явная демонстрация.
  - **Эволюция (F6)**: оттачивает параметры (1+1 Evolution Strategy).
- 🗺️ **Память всей трассы**: круг делится на 240 отрезков, на каждом запоминается
  кривизна, направление и **твоя траектория руля** → бот едет по выученному кругу,
  а не только по тому, что видит сейчас.
- 🏁 **Круги и время**: отметка старт/финиш (F4), время круга, лучший круг.
- ⚠️ **Память corner cut**: запоминает, где срезал/вылетел, и **сам притормаживает**
  заранее в этих местах.
- 🛡️ **Антибаги**: не уходит в реверс, не флемит при потере трассы, тред-безопасный ввод.
- 🖥️ **GUI-приложение** с живой телеметрией + работа из консоли.

## Установка

Нужен Python 3.10+ на Windows.

```bash
pip install -r requirements.txt
```

## Запуск

- **С интерфейсом:** двойной клик по `Запустить-GUI.bat` (запросит права админа).
- **Из консоли:** `Запустить.bat`.

> Права администратора обязательны — иначе ввод не дойдёт до Roblox.

### Горячие клавиши (работают глобально, даже в игре)

| Клавиша | Действие |
|--------|----------|
| **F5** | Запись: едешь сам, бот запоминает |
| **F6** | Обучение (эволюция) |
| **F8** | Ехать самому |
| **F4** | Отметить старт/финиш круга |
| **F7** | Калибровка зрения (показывает, что видит) |
| **F9** | Выход |

### Рекомендуемый сценарий

1. Открой Formula Apex Racing, зайди в заезд.
2. Запусти приложение → подтверди UAC.
3. На линии старт/финиш нажми **F4**, проедь круг сам, на той же линии — **F4** ещё раз.
4. Покатай несколько кругов сам — бот учится (видно в GUI: «Трасса изучена %»).
5. Жми **F8** — поедет сам по твоей траектории и гоночной линии, улучшая время.

## Как это работает

```
window.py     → находит окно Roblox и его область на экране
vision.py     → сегментация асфальта (самокалибровка цвета) + восприятие сцены
                (near/mid/far/line/curv) + оценка скорости (оптический поток)
controller.py → ввод: руль курсором, газ/тормоз/ERS, чтение действий игрока
learning.py   → имитация манеры (gain, пороги) + эволюция параметров (мозг brain.json)
track.py      → карта круга по 240 отрезкам: кривизна, траектория игрока, срезы,
                время круга (track_map.json)
autopilot.py  → главный мозг: восприятие → руление/газ/тормоз/ERS → обучение
app.py        → GUI-панель с телеметрией
```

Управление рулём = смесь трёх источников: **зрение сейчас** + **гоночная линия
(апекс)** + **твоя запомненная траектория на этом участке круга**. Торможение —
по кривизне впереди (из зрения и из памяти трассы), с защитой от реверса.

## Конфигурация (`config.json`)

Ключевые параметры:

| Раздел | Параметр | Смысл |
|--------|----------|-------|
| `capture` | `mode`, `window_title` | захват окна Roblox |
| `keys` | `throttle/brake/ers/drs/reset` | привязки (mouse:left и т.п.) |
| `vision` | `adaptive`, `color_tol`, `band_top/bottom` | зрение и обзор |
| `behavior` | `human_line_weight` | насколько копировать твою траекторию |
| `behavior` | `map_weight`, `map_lookahead` | вес памяти трассы |
| `behavior` | `brake_floor`, `max_brake_time` | защита от лишнего тормоза/реверса |
| `behavior` | `mem_frames` | сколько кадров твоей езды держать |
| `params` / `bounds` | стартовые значения и границы эволюции |

Данные обучения (`brain.json`, `demos.json`, `track_map.json`) создаются локально
и не входят в репозиторий. Удали их, чтобы начать с чистого листа.
